## Project Overview

FunBros is a fast-paced 2D platformer game developed using the Godot Engine and GDScript. The game focuses on smooth movement, precise timing, and engaging enemy interactions across **three fully playable levels**. Players can run, jump, roll, and navigate through colorful environments filled with traps, enemies, and checkpoints.

Each level introduces increased difficulty, new enemy placements, and more complex platforming challenges, delivering a complete and enjoyable platformer experience from start to finish.

---

## Key Features

- Classic 2D platformer gameplay  
- Smooth player movement and responsive controls  
- Three fully designed levels with progressive difficulty  
- Multiple enemy interactions with jump and roll mechanics  
- Life-based system with intelligent respawn behavior  
- Checkpoint-based progression system  
- Environmental hazards, traps, and pitfalls  
- Colorful and engaging level design  
- Complete and playable final game  

---

## Gameplay Mechanics

- Player can run, jump, and roll  
- Enemies can be defeated by:
  - Jumping on them  
  - Rolling into them  
- Falling into pits or direct enemy contact reduces player lives  
- Checkpoints save player progress within each level  
- Player respawns at the last checkpoint after losing a life  
- Game resets to the beginning when all lives are lost  

---

## Game Design Philosophy

FunBros is designed around simple mechanics combined with meaningful challenges. The gameplay emphasizes timing, movement precision, and player decision-making rather than complex controls.

The design focuses on fair enemy encounters, clear visual feedback, gradual difficulty progression across levels, and fast respawn cycles to keep the experience challenging yet fun.

---

## Technical Highlights

- Custom player movement logic for smooth and responsive controls  
- Physics-based enemy interaction handling  
- State-driven player actions (idle, run, jump, roll)  
- Modular and scalable script architecture  
- Checkpoint system implementation  
- Structured level design supporting multiple stages
